GitBook PDF Generator

A tool to convert GitBook-style markdown documentation into PDF.

Setup

npm install

Usage

There are two ways to generate PDFs:

  1. Using Puppeteer (recommended):
npm start
  1. Using markdown-pdf:
node generate-pdf.js

Both methods will generate output.pdf in the project root.

Project Structure

  • README.md - This documentation
  • chapters/ - Your markdown content
  • index.js - Main PDF generator (Puppeteer)
  • generate-pdf.js - Alternative PDF generator (markdown-pdf)

Adding Content

  1. Place your markdown files in the project
  2. Files will be processed in this order:
    • README.md first
    • Other files alphabetically
  3. Run the generator using one of the methods above

The generated PDF will include all markdown files with proper styling and formatting.
